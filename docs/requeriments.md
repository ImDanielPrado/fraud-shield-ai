# Requisitos - Fraud Shield AI

## Objetivo

O Fraud Shield AI é um sistema de análise de transações com foco em detecção de risco e possíveis fraudes.

O projeto foi desenvolvido com fins de estudo e demonstração de conceitos de Engenharia de Software, Arquitetura Backend e Inteligência Artificial.

## Funcionalidades do MVP

* Cadastro de usuários
* Cadastro de transações
* Cálculo de score de risco
* Geração de alertas
* Dashboard de métricas

## Funcionalidades Futuras

* Autenticação JWT
* Integração com IA local usando Ollama
* Histórico de análises
* Sistema de auditoria
* Dashboard avançado
