# Fraud Shield AI

Projeto de portfólio desenvolvido para demonstrar conceitos de Engenharia de Software, Arquitetura Backend e Integração com Banco de Dados.

## Tecnologias

* Node.js
* Express
* PostgreSQL
* Prisma ORM
* JavaScript

## Funcionalidades atuais

* Health Check da API
* Cadastro de usuários
* Persistência de dados com Prisma
* Migrations versionadas

## Arquitetura

Route → Controller → Service → Repository → Prisma → PostgreSQL

## Próximos passos

* Autenticação JWT
* CRUD de transações
* Motor de análise de risco
* Sistema de alertas
* Integração com IA local (Ollama)
